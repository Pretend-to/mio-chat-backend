## 🏫 典型教学场景

### 📌 场景一：课后作业即时批改

**学生**提交冒泡排序代码：

```python
def sort(arr):
    for i in range(len(arr)):
        for j in range(len(arr)):
            if arr[i] < arr[j]:
                arr[i], arr[j] = arr[j], arr[i]
    return arr
```

**教学助手**审查发现并执行验证：
- ⚠️ 内层循环多遍历了一次，可优化为 `range(len(arr)-i-1)`
- ⚠️ 原始数组被修改（副作用），建议传副本
- ✅ 执行验证排序结果正确
- 📊 输出排序前后对比图

### 📌 场景三：算法竞赛/考试训练

```python
# 学生提交：两数之和
def two_sum(nums, target):
    for i in range(len(nums)):
        for j in range(i+1, len(nums)):
            if nums[i] + nums[j] == target:
                return [i, j]
```

**教学助手**审查：
- ✅ 正确性：通过
- ⚡ 性能：O(n²) → 可优化为 O(n) 哈希表解法
- 📖 可读性：建议添加类型提示
- 🚀 沙盒执行测试多组用例验证正确性
