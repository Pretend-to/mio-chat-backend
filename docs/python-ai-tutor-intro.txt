一、平台背景：蜜柚（MioChat）

蜜柚是一款企业级多协议 AI 对话平台与 Agent 操作系统，由 Node.js 构建，具备高性能、模块化的特性。她的核心能力不止于对话——她拥有一套三位一体的扩展架构：

  - Skills（能力包）：自然语言定义的专家级能力，让 Agent 掌握垂直领域知识
  - MCP（标准协议）：毫秒级接入外部生态工具，极速扩充工具箱
  - Plugins（原生插件）：高性能业务扩展，支持文件编辑、终端交互等功能

蜜柚的创新亮点：Agent 可以自主编写、测试并加载新插件，实现功能的自我迭代；内置持久化任务调度引擎，支持长时周期性任务；生产级安全性保障，敏感配置自动脱敏。

正是基于蜜柚强大的 Agent 扩展能力，Python + AI 教学助手应运而生。

在 Python 课程教学中，学生常面临几个核心痛点：

  [痛点2] 代码能跑但不规范

    传统解法：本地配环境折腾半天
    我们的方案：沙盒内置 matplotlib + 中文支持，一键出图

    传统解法：脑内模拟容易出错

  [痛点5] 中文图表乱码
    传统解法：字体配置劝退初学者
    我们的方案：自动注入中文字体，无需任何配置

一句话总结：学生写完 Python 代码，交给教学助手，秒级获得专业审查报告 + 执行结果 + 可视化图表，让学习反馈从天级压缩到秒级。


三、核心能力详解

（一）8 维静态代码审查

不执行代码就能发现问题，覆盖教学中最关心的维度：

  1. 正确性 —— 逻辑错误、边界条件、类型假设
  2. 可读性 —— 命名规范、注释质量、函数长度
  3. 性能 —— 时间复杂度、不必要的循环
  4. 安全性 —— eval/exec、注入风险
  5. PEP 8 风格 —— 缩进、命名约定、导入顺序
  6. 健壮性 —— 异常处理、资源释放
  7. 模块化 —— 函数拆分、可测试性
  8. 文档 —— docstring、类型提示

教学意义：学生不仅知道代码对不对，更明白好代码长什么样。


（二）安全沙盒执行

内置 Python 沙盒环境，预装了教学最常用的库：

  科学计算  —— numpy、scipy         —— 数组运算、数值分析
  数据可视化 —— matplotlib、wordcloud —— 图表绘制、词云生成
  数据处理  —— pandas              —— 数据清洗、统计分析
  图像处理  —— Pillow              —— 图像生成、编辑
  中文分词  —— jieba               —— 中文文本处理

无需 pip install，学生提交代码即可运行，专注于编程逻辑本身。


（三）中文可视化智能适配

我们解决了 Python 教学中最麻烦的中文乱码问题。学生只需要写：

  import matplotlib.pyplot as plt

教学价值：学生在初学阶段不会被字体配置这类环境问题劝退，聚焦于编程逻辑本身。

四、典型教学场景

场景一：课后作业即时批改

学生提交冒泡排序代码：

  def sort(arr):
      for i in range(len(arr)):
          for j in range(len(arr)):
              if arr[i] < arr[j]:
                  arr[i], arr[j] = arr[j], arr[i]
      return arr

教学助手审查发现并执行验证：
  - 内层循环多遍历了一次，可优化为 range(len(arr)-i-1)
  - 原始数组被修改（副作用），建议传副本
  - 执行验证排序结果正确
  - 输出排序前后对比图


场景二：数据分析可视化作业

学生提交某班级成绩分析代码，自动获得：
  - 代码审查：变量命名建议、pandas 最佳实践
  - 沙盒执行：直接生成柱状图、饼图、箱线图
  - 中文完美显示：无需学生配置字体


场景三：算法竞赛/考试训练

学生提交两数之和：

  def two_sum(nums, target):
      for i in range(len(nums)):
          for j in range(i+1, len(nums)):
              if nums[i] + nums[j] == target:
                  return [i, j]

教学助手审查：
  - 正确性：通过
  - 性能：O(n^2) 可优化为 O(n) 哈希表解法
  - 可读性：建议添加类型提示
  - 沙盒执行测试多组用例验证正确性
